IperUnits

IperUnits (github) is a suite of multi-purpose Java libraries designed to meet various application needs. It includes an ecosystem of UltraLibs, a collection of generic tools to simplify application development. They form the foundation of a suite of apps, such as UltraServices.

IperUnits originated from the ashes of UltraUnits in 2008.

UltraServices

The UltraServices are a collection of services that can be installed with a single command on the cloud. They include features for messaging, data storage and analysis, AI tools, LLM routing. Rag Tools have been built on top of QuickRag, and then expanded and implemented also within Vault.

UltraServices provides complete local LLM architectures and a set of ready-to-use features, such as voice cloning, transcription, interpretation, and analysis.

Slim JEC & Suite

SlimJEC is an UltraLight Pure Javascript Client for basic access to ElasticSearch instances. No fancy frameworks, No Node.js, No Python. Just your browser. Experience SlimJEC with a SlimSuite of powerful standalone one-page tools developed for chatting.

Vault Project

Vault is a digital sanctuary right in the palm of your hands. It helps you create a digital survival manual, a Knowledge Vault that can be accessed offline, in a world where basic services are no longer guaranteed. The goal is to store it on an energy-independent device, or even a wearable wrist Pip-Boy. The knowledge base comes with fast search capabilities, powered by FTS, embeddings, and LLM technologies.

Web AI Assistants

Built to harness the power of browser technologies, they'll help you manage the overwhelming overload of information we are exposed to daily, and assist you with everyday tasks. Meet Whisperer, an intelligent speech recognition assistant that acts as your real-time conversation companion providing context-aware suggestions, interactive feedback, and BrowserMcWingspan, a lightweight, hackable browser assistant that lives in your sidebar. Get AI-powered browsing superpowers without the complexity.

SuperReact

SuperReact is a suite of libraries and components for React that makes it easy to create websites with seamless navigation, form submission workflows, and multi-app applications. It allows you to add localization features, mock servers, multimedia text enhancement, and more. See examples like AutoForm, ReactCV DelgadoMessenger, ShoolTrainer TreeMeth and more, all built on top of that.
